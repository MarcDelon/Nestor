Webographie du Projet SafeTrip

Marketplace de transport routier et de suivi de colis – ressources techniques et documentations

Généré le:

1. Technologies Fondamentales & Frameworks

React (v19.0)

Description : Bibliothèque JavaScript de référence pour la création d'interfaces utilisateur réactives et modulaires.

Lien : https://react.dev/

Next.js (v16.2 - Turbopack)

Description : Framework React de production avec routage basé sur les dossiers (App Router), rendu côté serveur (SSR), génération de pages statiques (SSG) et compilation ultra-rapide Turbopack.

Lien : https://nextjs.org/docs

Node.js & Express

Description : Environnement d'exécution JavaScript côté serveur et micro-framework minimaliste pour le développement d'APIs REST modulaires et performantes.

Liens :
- https://nodejs.org/
- https://expressjs.com/

TypeScript (v5.0)

Description : Sur-ensemble typé de JavaScript garantissant la robustesse du code, l'auto-complétion intelligente et la sécurité des types à la compilation sur le frontend et le backend.

Lien : https://www.typescriptlang.org/docs/

2. Base de Données & Services Backend (BaaS)

Supabase & PostgreSQL

Description : Plateforme Backend-as-a-Service (BaaS) open-source basée sur PostgreSQL. Utilisée pour l'hébergement de la base de données relationnelle, l'authentification sécurisée, la gestion des sessions, et le requêtage asynchrone via le client unifié.

Lien : https://supabase.com/docs

PostgreSQL Schema & RLS (Row Level Security)

Description : Documentation sur les meilleures pratiques de conception de schémas relationnels SQL et de sécurisation fine des données via les politiques RLS.

Lien : https://www.postgresql.org/docs/

3. Génération & Lecture de Codes QR (Système de Traçabilité)

API QR Code Generator (goqr.me)

Description : Service cloud gratuit de génération instantanée de codes QR. Utilisé pour créer dynamiquement les codes QR uniques des billets voyageurs et des étiquettes de colis à partir de données sérialisées en temps réel.

Lien : https://goqr.me/api/

jsQR (v1.4)

Description : Bibliothèque JavaScript d'analyse d'images pure permettant de détecter et de décoder en temps réel les codes QR à partir de flux vidéo de caméras (webcam). Utilisée pour la simulation du scanner en gare dans le tableau de bord de l'agence.

Lien : https://github.com/cozmo/jsQR

4. Génération de Documents PDF & Rendu Graphique

jsPDF (v4.2)

Description : Bibliothèque JavaScript côté client pour la génération de documents PDF à partir de scripts. Utilisée pour éditer et télécharger les billets de voyage et les étiquettes de bagages officiels de l'agence.

Lien : https://rawgit.com/MrRio/jsPDF/master/docs/index.html

html2canvas (v1.4)

Description : Bibliothèque de capture d'écran d'éléments DOM HTML. Elle convertit le rendu HTML d'un ticket en une image matricielle de haute précision, qui est ensuite vectorisée dans le document PDF via jsPDF.

Lien : https://html2canvas.hertzen.com/

5. Intégrations Tierces & Utilitaires

Nodemailer & SMTP Transport

Description : Module d'envoi d'e-mails pour Node.js. Utilisé dans le backend pour alerter automatiquement et instantanément l'expéditeur et le destinataire par e-mail lors de la livraison d'un colis ou courrier en gare.

Lien : https://nodemailer.com/

Google Fonts (Montserrat, DM Sans, Syne)

Description : Répertoire de polices typographiques modernes et optimisées pour le web, structurant l'identité visuelle haut de gamme et harmonieuse de SafeTrip.

Lien : https://fonts.google.com/

Smartsupp Integration

Description : Service de chat en ligne pour interagir en temps réel avec les clients et stocker les sessions de demandes spéciales dans l'historique de l'administration.

Lien : https://www.smartsupp.com/
